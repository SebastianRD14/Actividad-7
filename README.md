# 📱 Actividad 7 - Intents y Comunicación entre Apps

## ❓ Preguntas y Respuestas

### 🔗 Pregunta 1: Aprovechamiento de funcionalidades

**¿Cómo se pueden aprovechar las funcionalidades de otras aplicaciones para mejorar la experiencia del usuario en una aplicación propia?**

**Respuesta:** Usando intents y diferentes funciones de estos para poder acceder a otra aplicaciones y las funciones de las mismas para no tener que hacer ninguna de estas desde cero.

### 🚀 Pregunta 2: Intents para abrir aplicaciones

**¿Cómo se pueden utilizar Intents para abrir otras aplicaciones de forma segura y eficiente?**

**Respuesta:** Verificando que ue la aplicación exista antes de abrirla para que no exista ningun fallo critico en la aplicación.

### 🔄 Pregunta 3: Compartir datos entre aplicaciones

**¿Cómo se pueden utilizar Intents para compartir datos y acciones entre diferentes aplicaciones?**

**Respuesta:** Usando Intents implícitos con acciones como ACTION_SEND, para compartir texto, imágenes o enlaces entre aplicaciones.

### 💭 Reflexión Personal

**Reflexión del tema (mínimo 50 palabras):**
Gracias a los intents pude ver como poder acceder a diferentes activities y aplicaciones para poder acceder a funciones de estas sin tener que hacer manualmente cada una, sino que usando las herramientas creadas poder hacer una mejor aplicación, como también me di cuenta de lo interesante y curioso que esa función puede hacer para el proyecto.
